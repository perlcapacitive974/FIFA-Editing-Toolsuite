## 📋 Best Practices

Keep your game files safe by following these simple habits:

* Backup your game save files before you use any modding tools. You can find these in your Documents folder under the game name.
* Modify one item at a time. This makes it easier to identify the source if the game displays an error or fails to load.
* Check the release page frequently for updates. New patches from the game developer often require matching updates for the toolsuite to maintain full compatibility.
* Use clean files for your textures. Large, unoptimized images can impact performance during matches.

## ❓ Frequently Asked Questions

**Will this tool get me banned from online play?**
The tool modifies local files only. While it works for career mode, rosters, and visual assets, you should disable all mods before you attempt to connect to official online servers. Modified files create a mismatch with server-side data and prevent you from joining online matches.

**Can I use these assets with other games?**
This tool connects specifically to games built on the Frostbite engine. Attempting to force these assets into games using different engines will result in errors or crashes.

**The tool fails to launch. What should I do?**
Ensure you have the latest .NET Framework installed from the Microsoft website. If you still have trouble, check that your antivirus software is not blocking the application from accessing the game files.

**How do I install custom player faces?**
You need the 3D model path and the accompanying texture file. Load the model into the "Face Editor" tab of the toolsuite, map the textures to the correct file nodes, and save your progress. The tool handles the file compression for the game engine automatically.
